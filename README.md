MySublimeTextStuff
==================

## Prerequisites:
- [jshint](http://www.jshint.com/install/)
- [flake8](http://flake8.readthedocs.org/en/latest/)

## Installation
Pull repo into User folder: 
```
~/Library/Application Support/Sublime Text 3/Packages/User
```
